## Cursor Assets

The cursor icons used in this application are from SVG Repo (svgrepo.com) and are licensed under CC0 (Creative Commons Zero v1.0 Universal).

- **arrow.svg** - Arrow cursor (default pointer)
- **ibeam.svg** - I-Beam cursor (text selection)
- **wait.svg** - Wait cursor (hourglass/busy)
- **cross.svg** - Crosshair cursor
- **resize_h.svg** - Horizontal resize cursor
- **resize_v.svg** - Vertical resize cursor
- **resize_nwse.svg** - Diagonal resize cursor (NW-SE)
- **resize_nesw.svg** - Diagonal resize cursor (NE-SW)
- **move.svg** - Move cursor (4-way arrow)
- **hand.svg** - Hand cursor (link pointer)
- **no.svg** - Not-allowed cursor

**License**: CC0 1.0 Universal (Public Domain Dedication)

These assets have been converted to Android Vector Drawables and styled with white fill and black outline for visibility on all backgrounds. The original SVG files are located in `assets/cursors/`.
